# To-Do List

Une application web simple pour gérer ses tâches quotidiennes.

## Technologies utilisées
- PHP
- CSS
- MySQL
- Git & GitHub

## Installation
1. Cloner le dépôt :
   git clone https://github.com/bobche04/tp1-git-github.git
2. Déplacer le dossier dans C:\laragon\www\
3. Démarrer Laragon
4. Ouvrir le navigateur et aller sur : http://todo-list.test

## Fonctionnalités
- Ajouter une tâche
- Afficher la liste des tâches
- Supprimer une tâche (à venir)
